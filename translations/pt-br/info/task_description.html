Comecei a alimentar um pombo. Um minuto depois, chegaram mais dois voando, e um minuto depois chegaram mais três. Em seguida, 4, e assim por diante (Ex 1 2 3 4 + ...). Uma porção de comida alimenta um pombo por um minuto. Se a porção de comida não é suficiente para todos os pássaros, os pombos que chegaram mais cedo comem primeiro. Os pombos são animais famintos e vão comer sem saber quando parar. Se eu tiver N porções de comida, quantos pombos serão alimentados com pelo menos uma porção de trigo?

Entrada: O numero de porções de trigo como um número inteiro positivo (int).

Saída: O número de pombos alimentados como um inteiro (int).

Exemplo: 

checkio(1) == 1
checkio(2) == 1
checkio(5) == 3
checkio(10) == 6

Como é utilizado: Esta tarefa mostra como podemos usar programação para modelar várias situações. É claro que o modelo tem limitações, mas muitas vezes nós não precisamos de um modelo perfeito.

Pré-condição: 0 < N < 105.
