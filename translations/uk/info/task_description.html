Я почав годувати одного з голубів. Через хвилину прилетіло ще два, і ще через хвилину прилетіло ще три голуби. Потім 4 і так далі (Напр.: 1 +2 +3 +4 + ...). Одної порції корму вистачає одному голубу на хвилину. У разі, якщо їжі не вистачає на всіх птахів, тоді спочатку їдять ті голуби, що прилетіли раніше. Голуби - це вічно голодні птахи і вони будуть їсти та їсти без зупинки. Якщо у мене є N порцій корму, то скільки голубів я зможу погодувати хоча б по разу?

Вхідні дані: Кількість порцій корму, як ціле число (int).

Вихідні дані: Кількість нагодованих голубів, як ціле число (int).

Приклади: 

checkio(1) == 1
checkio(2) == 1
checkio(5) == 3
checkio(10) == 6

Як це використовується: У цій задачі ми бачимо, як можна використовувати програмування для моделювання ситуацій. Звичайно, будь-яка модель має свої обмеження і наближення, але найчастіше нам і не потрібна ідеальна модель.

Передумови: 0 < N < 105.
